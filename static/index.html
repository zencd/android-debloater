Using this section:
  1. Connect your Android device to the PC
  2. Allow the connection on Android
  3. In the package filter, select "Safe to debloat"
  4. Mark some unnecessary packages as "Debloat"
  5. Click the "Debloat" button
Preparing Android:
  1. Become a developer, tapping OS build number a lot
  2. Developer options:
    • Enable "USB debugging"
Использование этого раздела:
  1. Подключите Android-устройство к ПК
  2. Разрешите подключение на Андроиде
  3. В фильтре пакетов выберите "Безопасно удалять"
  4. Пометьте лишние пакеты как "Удалить"
  5. Нажмите кнопку "Очистить"
Подготовка Android:
  1. Станьте разработчиком, многократно нажав на номер сборки ОС
  2. Параметры разработчика:
    • Разрешите "Отладку по USB"

Using this section:
  1. Connect your Android device #1 to the PC
  2. Click menu "Backup / Apps"
  3. Connect your Android device #2 to the PC
  4. Refresh page
  5. Click menu "Restore / Apps"
Preparing Android:
  1. Developer options:
    • Enable "USB Debugging Security Settings" (Xiaomi)
    • Enable "Install via USB" (Xiaomi)
    • Disable permission monitoring (Color/Oxygen)
    • You may be required to insert a SIM (Xiaomi)
    • You will be required to confirm every app install (Xiaomi)

Использование этого раздела:
  1. Подключите к ПК Android-устройство №1
  2. Нажмите пункт меню "Бэкап / Приложения"
  3. Обновите страницу
  4. Подключите к ПК Android-устройство №2
  5. Нажмите пункт меню "Восстановление / Приложения"
Подготовка Android:
  1. Параметры разработчика:
    • Разрешите "Отладку по USB, Настройки безопасности" (Xiaomi)
    • Разрешите "Установку через USB" (Xiaomi)
    • Отключите мониторинг разрешений (Color/Oxygen)
    • Может потребоваться вставить SIM-карту (Xiaomi)
    • Установка каждого приложения из бэкапа потребует подтверждения на девайсе (Xiaomi)
